# ReviewFilter
面向短文本的网络言论智能过滤系统实现
=======================
系统架构部分
-----------
1：客户端仅支持安卓，是一个面向客户的APP。客户可以使用获取，浏览，评论新闻等功能，以及点赞，收藏，分享等<br>
2:管理端采用Spring+SpringMVC+Mybaties搭建，管理员可以管理，发布新闻，管理评论，管理用户信息等<br>
3:后端嵌入智能过滤评论算法，对用户上传的评论进行分类判断，决定是否发布<br>
